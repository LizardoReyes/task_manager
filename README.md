### Guía de uso

### Compilación y ejecución  
`cargo build`

### Añadir una tarea  
`cargo run -- add "Realizar la EU"`

### Listar tareas  
`cargo run -- list`

### Marcar una tarea como completada
`cargo run -- mark-done 1`

### Desmarcar una tarea
`cargo run -- unmark-done 1`

### Eliminar una tarea por ID  
`cargo run -- delete 1`

### Eliminar todas las tareas  
`cargo run -- delete-all`
